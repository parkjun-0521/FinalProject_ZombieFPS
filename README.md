  ## 1. 개요 
  프로젝트 명 : Project Z. Zoombie
  
  장르 : 1인칭 FPS 게임 
  
  배경 : 좀비, 아포칼립스
  
  플랫폼 : PC, VR

  소개 : 플레이어는 다양한 아이템을 습득하여 수 많은 좀비에게서 생존해야 하며, 특정 임무를 수행하여 최종 스테이지까지 이동하여 탈출하는 게임입니다.
